# 🌟 Premium Image Gallery

A modern and responsive **Image Gallery Website** built using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Frontend Development Internship**.

This project features a stylish dashboard-style UI with smooth hover effects, lightbox image preview, navigation controls, and responsive design for all screen sizes.

---

## 🚀 Features

✅ Responsive Image Gallery Layout  
✅ Modern Dashboard UI Design  
✅ Hover Zoom Effects & Smooth Transitions  
✅ Lightbox Image Preview  
✅ Next / Previous Navigation Buttons  
✅ Mobile Friendly Design  
✅ Stylish Glassmorphism Cards  
✅ Interactive User Experience

---

- HTML5
